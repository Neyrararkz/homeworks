Часть 1. Теоретическая

1. LocalStorage — это хранилище внутри самого браузера.
2. SessionStorage, в оличие от localStorage, хранит данные только до момента закрытия вкладки браузера.
3. Потому что localStorage нужен, чтобы хранить данные неограниченное количество времени

• Сохранение значения:
localStorage.setItem("username", "Alice")
• Получение значения:
const user = localStorage.getItem("username")
console.log(user)
• Удаление значения:
localStorage.removeItem("username")
